项目整体思路
任务定义
输入一张图片 + 用户评论文本，输出一个 10维概率分布（对应1~10分），预测图像的美学质量。

核心设计理念
骨干全冻结，只训练轻量适配层。 CLIP、BERT、PARN 三个预训练模型的参数全部不动，通过少量可学习的 Query 和 Adapter 从中提取美学相关特征。总参数 612M，可训练只有 176M。

四阶段架构
阶段1 — 多尺度特征提取

视觉：冻结 CLIP ViT-L/14，取第 6/12/18/24 层的中间隐藏状态（1024维），各自用一个可训练投影层压到 768 维，得到 4 级视觉特征 V1~V4，每级形状 (B, 257, 768)
文本：冻结 BERT，取第 3/6/9/12 层，每层后接一个可训练轻量 Adapter（残差结构），得到 4 级文本特征 T1~T4，每级形状 (B, seq, 768)
两个模态各出 4 个层级，形成层级对齐的多尺度表示。

阶段2 — 弱交互式融合（×4层）

每个层级有独立的可学习 Query（2个视觉 + 2个文本），对每一层做三步：


Step1 模态内：VQ = CrossAttn(VQ, V)   TQ = CrossAttn(TQ, T)
Step2 跨模态：VQ_hat = CrossAttn(VQ, T)   TQ_hat = CrossAttn(TQ, V)
Step3 均值：  VQ_final = (VQ + VQ_hat) / 2
"弱交互"的含义是：先各自从本模态提取，再互相看对方，最后取均值融合，而不是直接强制合并。4个层级各自独立处理，保留了多尺度信息。

阶段3 — 层级聚合 + 动态属性推理

层级聚合：4层的 Query 输出分别做 mean pooling 压成 token，加一个可学习 global token，过 2层 Self-Attention，取 global token 位置的输出作为最终视觉/文本全局向量各一个 (B, 768)
属性推理：冻结的 PARN（ResNet-50）提取图像全局特征 g(B,2048) 和 11个美学属性分数，每个属性拼接 [g, score_i] 投影成一个 token，共 11个 attribute tokens。然后用多模态 Query 对这 11个 tokens 做 CrossAttn 精炼
阶段4 — 预测头


[VQ_global, TQ_global] → concat → Linear(1536→768) → Linear(768→10) → Softmax
输出 10维概率分布。

损失函数

Loss = EMD₂  +  0.2 × MSE(均值分)  +  0.1 × PairwiseRankingLoss
EMD（Earth Mover's Distance）：主损失，衡量预测分布和真实分布的距离，比交叉熵更适合有序分布
MSE on mean：辅助，直接约束预测均值分和真实均值分的误差
Ranking Loss：辅助，对 batch 内所有图片对，惩罚预测排序和真实排序不一致的情况
