\begin{abstract}
Vision-language models have shown strong potential in image aesthetic assessment (IAA), yet effectively exploiting user comments with limited trainable parameters remains challenging. In this letter, we introduce LQ-ICIF, a parameter-efficient multimodal framework for aesthetic score distribution prediction. To capture aesthetic cues from images and comments, LQ-ICIF integrates two key components: 1) learnable query-guided intra-cross interaction fusion, which extracts hierarchical visual-comment representations from frozen CLIP and BERT backbones, and 2) attribute-guided representation refinement, which incorporates photographic aesthetic attributes predicted by a frozen PARN model as structured priors. Experiments show that LQ-ICIF achieves an SRCC of 0.9145, PLCC of 0.9213, classification accuracy of 87.51\%, and EMD of 0.0403, demonstrating accurate and efficient multimodal aesthetic distribution prediction.
\end{abstract}

Image aesthetic assessment, user comments, multimodal learning, score distribution prediction, attribute-guided refinement.

\section{Introduction}

\IEEEPARstart{I}{mage} aesthetic assessment (IAA) aims to automatically predict human aesthetic perception of images and has attracted increasing attention due to applications in image retrieval, photo recommendation, automatic enhancement, and social media content management [1], [2]. Unlike conventional image quality assessment, IAA is affected by composition, lighting, color harmony, semantic content, photographic style, and subjective preference. Therefore, accurately predicting aesthetic score distributions remains challenging.

\begin{figure}[!t]
\centering
\includegraphics[width=\columnwidth]{method_comparison.png}
\caption{Comparison of representative image aesthetic assessment paradigms. Unlike visual-only and coarse auxiliary-fusion methods, LQ-ICIF exploits multi-level image-comment interaction and attribute-guided refinement for aesthetic score distribution prediction.}
\label{fig:method_comparison}
\end{figure}

Subjective evaluation methods are reliable but costly, motivating objective IAA models that automatically predict human aesthetic opinions. Early methods rely on handcrafted photographic descriptors, while recent convolutional and transformer-based models learn aesthetic representations directly from images [2], [3]. Distribution-based methods, such as NIMA, further predict rating distributions and optimize the Earth Mover's Distance (EMD), offering a more faithful description of subjective opinions than scalar regression [3]. However, most existing methods still depend primarily on visual content, while perceptual cues expressed in user comments are largely ignored.

To exploit such complementary information, several studies have explored image-comment modeling for IAA. User comments can explicitly describe blur, lighting, composition, color tone, subject emphasis, and emotional impression. For example, VILA learns aesthetic representations from image-comment pairs via vision-language pretraining [5], while MMLQ extracts multimodal aesthetic features from images and comments using learnable queries with frozen encoders [6]. Although effective, existing multimodal IAA methods still face three limitations. First, single-layer or final-layer features may overlook hierarchical aesthetic perception. Second, direct cross-modal fusion may introduce noisy interactions when comments are subjective, incomplete, or irrelevant. Third, structured aesthetic attributes are not fully exploited.

To address these challenges, we propose LQ-ICIF, a parameter-efficient multimodal framework for aesthetic score distribution prediction. As shown in Fig.~\ref{fig:method_comparison}, the proposed method keeps CLIP, BERT, and PARN frozen, and introduces lightweight projections, adapters, and learnable queries for task-specific adaptation [7]--[9]. Multi-scale visual features from CLIP and multi-level textual features from BERT are extracted to capture hierarchical cues. A weak cross-modal interaction module then preserves modality-specific information while modeling visual-textual complementarity. In addition, attribute-aware reasoning introduces structured photographic attribute tokens to guide prediction.

The main contributions of this work are summarized as follows:
\begin{itemize}
\item We propose a parameter-efficient multimodal IAA framework with frozen pretrained visual, textual, and attribute models.
\item We introduce a learnable query-guided intra-cross interaction mechanism to capture hierarchical visual-textual cues while reducing noisy over-fusion.
\item We incorporate attribute-guided representation refinement for robust distribution-level prediction.
\end{itemize}

The remainder of this paper is organized as follows. Section II details the proposed LQ-ICIF framework. Section III presents the experimental results. Finally, Section IV concludes the paper.

\section{Method}

\subsection{Background}

We use frozen CLIP, BERT, and PARN models [10]--[12] as visual, textual, and photographic attribute priors, respectively, and train only lightweight adaptation modules for image-comment aesthetic distribution prediction. This design keeps the framework parameter-efficient while preserving the general representation ability of pretrained models.

Given the $i$-th image $I_i$ and its corresponding comment $C_i$, CLIP and BERT provide multi-level features:
\begin{equation}
\{\mathbf{V}_i^l\}_{l=1}^{L}=E_v(I_i), \quad
\{\mathbf{T}_i^m\}_{m=1}^{M}=E_t(C_i).
\end{equation}

The PARN branch extracts photographic attribute information $\mathbf{A}_i=E_a(I_i)$. Since these representations come from different backbones, we project them into a unified latent space:
\begin{equation}
\tilde{\mathbf{V}}_i^l=P_v^l(\mathbf{V}_i^l), \quad
\tilde{\mathbf{T}}_i^m=P_t^m(\mathbf{T}_i^m), \quad
\tilde{\mathbf{A}}_i=P_a(\mathbf{A}_i).
\end{equation}

\subsection{Problem Formulation}

\begin{figure*}[!t]
\centering
\includegraphics[width=\textwidth]{lq_icif_overview.png}
\caption{Framework of the proposed LQ-ICIF. It integrates (1) learnable query-guided intra-cross interaction fusion, modeling hierarchical image-comment relations with lightweight trainable queries; (2) global token-guided hierarchical aggregation, summarizing multi-level visual and textual cues into a compact multimodal representation; and (3) attribute-guided representation refinement, injecting PARN-based aesthetic attribute priors before distribution prediction.}
\label{fig:lqicif_overview}
\end{figure*}

As illustrated in Fig.~\ref{fig:lqicif_overview}, LQ-ICIF consists of three main modules: learnable query-guided intra-cross interaction fusion, global token-guided hierarchical aggregation, and attribute-guided representation refinement. Given an image-comment pair $(I_i,C_i)$, the goal is to predict a discrete aesthetic score distribution:
\begin{equation}
\hat{\mathbf{y}}_i=f(I_i,C_i),
\end{equation}
where $\hat{\mathbf{y}}_i\in\mathbb{R}^{K}$ and $K=10$ for AVA. The ground-truth distribution $\mathbf{y}_i$ satisfies $\sum_{k=1}^{K}y_i^k=1$. The mean aesthetic score is computed as
\begin{equation}
\label{eq:mean_score}
\hat{s}_i=\sum_{k=1}^{K} k\hat{y}_i^k.
\end{equation}
The following subsections describe the three modules and the final prediction head.

\subsection{Learnable Query-Guided Intra-Cross Interaction Fusion}

For the $l$-th visual-textual feature pair $(\tilde{\mathbf{V}}_i^l,\tilde{\mathbf{T}}_i^l)$, two learnable query sets, $\mathbf{Q}_{l,0}^{V}$ and $\mathbf{Q}_{l,0}^{T}$, are used to extract compact task-specific tokens. They first attend to their own modality:
\begin{equation}
\begin{aligned}
\mathbf{Q}_{l}^{V}
&=\mathrm{Attn}(\mathbf{Q}_{l,0}^{V},\tilde{\mathbf{V}}_i^l,\tilde{\mathbf{V}}_i^l),\\
\mathbf{Q}_{l}^{T}
&=\mathrm{Attn}(\mathbf{Q}_{l,0}^{T},\tilde{\mathbf{T}}_i^l,\tilde{\mathbf{T}}_i^l).
\end{aligned}
\end{equation}
The intra-modal step keeps modality-specific aesthetic evidence, such as visual composition or subjective textual impressions. Then, cross-modal attention exchanges complementary image-comment cues without directly merging all tokens:
\begin{equation}
\begin{aligned}
\hat{\mathbf{Q}}_{l}^{V}
&=\mathrm{Attn}(\mathbf{Q}_{l}^{V},\tilde{\mathbf{T}}_i^l,\tilde{\mathbf{T}}_i^l),\\
\hat{\mathbf{Q}}_{l}^{T}
&=\mathrm{Attn}(\mathbf{Q}_{l}^{T},\tilde{\mathbf{V}}_i^l,\tilde{\mathbf{V}}_i^l).
\end{aligned}
\end{equation}
The resulting cross-attended queries are added back to the intra-modal queries to obtain $\mathbf{F}_{l}^{V}$ and $\mathbf{F}_{l}^{T}$. Applying this block to multiple layers captures hierarchical image-comment interactions while keeping the pretrained encoders frozen.

\subsection{Global Token-Guided Hierarchical Aggregation}

Different feature levels capture different aesthetic cues, so simply using the final layer may discard useful information. We first mean-pool $\mathbf{F}_{l}^{V}$ and $\mathbf{F}_{l}^{T}$ into layer descriptors $\mathbf{z}_{l}^{V}$ and $\mathbf{z}_{l}^{T}$. The descriptors are then concatenated with learnable global tokens and processed by self-attention:
\begin{equation}
\begin{aligned}
\mathbf{S}^{V}
&=\mathrm{SA}([\mathbf{g}_{0}^{V};\mathbf{z}_{1}^{V};\ldots;\mathbf{z}_{L}^{V}]),\\
\mathbf{S}^{T}
&=\mathrm{SA}([\mathbf{g}_{0}^{T};\mathbf{z}_{1}^{T};\ldots;\mathbf{z}_{L}^{T}]).
\end{aligned}
\end{equation}
The updated global tokens serve as modality-level summaries:
\begin{equation}
\mathbf{F}_{\mathrm{cat}}=[\mathbf{h}^{V};\mathbf{h}^{T}], \quad
\mathbf{h}^{V}=\mathbf{S}^{V}_{0},\ \mathbf{h}^{T}=\mathbf{S}^{T}_{0}.
\end{equation}
Thus, the model adaptively selects useful information from multiple feature levels and forms a compact multimodal representation for refinement.

\subsection{Attribute-Guided Representation Refinement}

To reduce the effect of incomplete or noisy comments, the frozen PARN branch provides structured aesthetic priors. Given $I_i$, PARN produces a global attribute feature $\mathbf{g}_i$ and attribute scores $\mathbf{s}_i=[s_i^1,\ldots,s_i^A]$, covering explicit photographic properties. Each score is combined with $\mathbf{g}_i$ and projected into an attribute token:
\begin{equation}
\mathbf{a}_i^j=P_r([\mathbf{g}_i;s_i^j]), \quad j=1,\ldots,A.
\end{equation}
The attribute tokens form $\mathbf{A}_i=[\mathbf{a}_i^1,\ldots,\mathbf{a}_i^A]$, which refine the aggregated multimodal representation by cross-attention:
\begin{equation}
\hat{\mathbf{F}}_i=\mathrm{CrossAttn}(\mathbf{F}_{\mathrm{cat}},\mathbf{A}_i,\mathbf{A}_i).
\end{equation}
In this way, the final representation is guided toward perceptually meaningful aesthetic dimensions before prediction, rather than relying only on implicit multimodal fusion.

\subsection{Aesthetic Score Distribution Prediction}

The refined representation $\hat{\mathbf{F}}_i$ is fed into an MLP head followed by softmax to estimate the full rating distribution:
\begin{equation}
\hat{\mathbf{y}}_i=\mathrm{Softmax}(\mathrm{MLP}(\hat{\mathbf{F}}_i)),
\end{equation}
The softmax output keeps the full uncertainty of human ratings instead of collapsing them into a single scalar during training. Following distribution-based IAA methods, the normalized ground-truth rating histogram $\mathbf{y}_i$ supervises $\hat{\mathbf{y}}_i$ with the Earth Mover's Distance (EMD):
\begin{equation}
\mathcal{L}_{\mathrm{emd}} =
\left(
\frac{1}{K}\sum_{k=1}^{K}
\left|
\mathrm{CDF}_{\hat{\mathbf{y}}_i}(k)-
\mathrm{CDF}_{\mathbf{y}_i}(k)
\right|^r
\right)^{1/r},
\end{equation}
where $\mathrm{CDF}_{\hat{\mathbf{y}}_i}(k)=\sum_{j=1}^{k}\hat{y}_i^j$, $\mathrm{CDF}_{\mathbf{y}_i}(k)=\sum_{j=1}^{k}y_i^j$, and $r=2$. The final mean score is computed by \eqref{eq:mean_score}.

\section{Experiments}
\label{sec:experiments}

\subsection{Experimental Protocols}

We conduct experiments on image aesthetic assessment (IAA) datasets: AVA and AADB. The current study reports results on AVA, which contains 222,383 training, 11,707 validation, and 19,805 test images in our split. Each image is annotated with a 10-bin rating histogram. Since LQ-ICIF uses image-comment pairs, we further employ the AVA-Comments file, which contains 3,602,206 comments for 255,530 images. AADB is reserved as an additional benchmark after its preprocessing protocol is finalized.

To evaluate IAA performance, we employ SRCC and PLCC to measure score-level monotonicity and linear consistency, ACC for binary high/low aesthetic classification, and EMD for distribution-level prediction quality. The ground-truth AVA distribution is directly normalized from the original rating histogram. Our method is implemented in PyTorch, and all experiments are conducted on a single NVIDIA GeForce RTX 5060 Ti GPU.

\subsection{Implementation Details}

Following the image-comment setting of LQ-ICIF, all comments matched to an image are concatenated as text input, while missing comments are replaced by a ``no comment'' token. Images are resized to $224\times224$ and augmented by random horizontal flipping during training. We use frozen CLIP ViT-L/14, BERT-base, and PARN backbones, and optimize only the lightweight adaptation, fusion, refinement, and prediction modules. The model is trained for 10 epochs using AdamW with learning rate $1\times10^{-4}$, batch size 16, and a loss combining EMD, mean-score MSE, and pairwise ranking terms.

\subsection{Performance Comparison}

We compare our method with visual-only and multimodal image aesthetic assessment methods on the AVA dataset, with the results summarized in Table~\ref{tab:main_comparison}. From this table, several observations can be drawn:
\begin{itemize}
\item Visual-only baselines show limited performance, suggesting that image content alone is insufficient for subjective aesthetic prediction.
\item LQ-ICIF achieves the best current results, with 0.9145 SRCC, 0.9213 PLCC, 87.51\% ACC, and 0.0403 EMD.
\item With frozen CLIP, BERT, and PARN backbones, the gain mainly comes from the proposed lightweight fusion, aggregation, and attribute refinement modules.
\end{itemize}

We further plan to assess the generalization ability of the proposed method on the AADB dataset, which provides human aesthetic annotations beyond the AVA rating-distribution setting. After finalizing the AADB preprocessing protocol and comparable baselines, the additional results will be reported to verify whether LQ-ICIF remains effective across different aesthetic assessment benchmarks.

The IEEE Graphics Analyzer enables authors to pre-screen their graphics for 
compliance with IEEE Transactions and Journals standards before submission. 
The online tool, located at
\underline{http://graphicsqc.ieee.org/}, allows authors to 
upload their graphics in order to check that each file is the correct file 
format, resolution, size and colorspace; that no fonts are missing or 
corrupt; that figures are not compiled in layers or have transparency, and 
that they are named according to the IEEE Transactions and Journals naming

For more information on using the Graphics Analyzer or any other graphics 
related topic, contact the IEEE Graphics Help Desk by e-mail at 
graphics@ieee.org.

\subsection{Submitting Your Graphics}
Because IEEE will do the final formatting of your paper,
you do not need to position figures and tables at the top and bottom of each 
column. In fact, all figures, figure captions, and tables can be placed at 
the end of your paper. In addition to, or even in lieu of submitting figures 
within your final manuscript, figures should be submitted individually, 
separate from the manuscript in one of the file formats listed above in 
Section \ref{formats}. Place figure captions below the figures; place table titles 
above the tables. Please do not include captions as part of the figures, or 
put them in ``text boxes'' linked to the figures. Also, do not place borders 
around the outside of your figures.

\subsection{Color Processing/Printing in IEEE Journals}
All IEEE Transactions, Journals, and Letters allow an author to publish 
color figures on IEEE Xplore\textregistered\ at no charge, and automatically 
convert them to grayscale for print versions. In most journals, figures and 
tables may alternatively be printed in color if an author chooses to do so. 
Please note that this service comes at an extra expense to the author. If 
you intend to have print color graphics, include a note with your final 
paper indicating which figures or tables you would like to be handled that 
way, and stating that you are willing to pay the additional fee.

\section{Conclusion}

A conclusion section is not required. Although a conclusion may review the main points of the paper, do not replicate the abstract as the conclusion. A conclusion might elaborate on the importance of the work or suggest applications and extensions. 

\section*{Acknowledgment}

The preferred spelling of the word ``acknowledgment'' in American English is without an ``e'' after the ``g.'' Use the singular heading even if you have many acknowledgments. Avoid expressions such as ``One of us (S.B.A.) would like to thank . . . .'' Instead, write “F. A. Author thanks ... .” In most cases, sponsor and financial support acknowledgments are placed in the unnumbered footnote on the first page, not here.

\section*{References and Footnotes}

\subsection{References}

References need not be cited in text. When they are, they appear on the line, in square brackets, inside the punctuation.  Multiple references are each numbered with separate brackets. When citing a section in a book, please give the relevant page numbers. In text, refer simply to the reference number. Do not use ``Ref.'' or ``reference'' except at the beginning of a sentence: ``Reference [3] shows . . . .'' Please do not use automatic endnotes in {\em Word}, rather, type the reference list at the end of the paper using the ``References'' style.

Reference numbers are set flush left and form a column of their own, hanging out beyond the body of the reference. The reference numbers are on the line, enclosed in square brackets. In all references, the given name of the author or editor is abbreviated to the initial only and precedes the last name. Use them all; use {\em et al.} only if names are not given. Use commas around Jr., Sr., and III in names. Abbreviate conference titles.  When citing IEEE transactions, provide the issue number, page range, volume number, year, and/or month if available. When referencing a patent, provide the day and the month of issue, or application. References may not include all information; please obtain and include relevant information. Do not combine references. There must be only one reference with each number. If there is a URL included with the print reference, it can be included at the end of the reference.

Other than books, capitalize only the first word in a paper title, except for proper nouns and element symbols. For papers published in translation journals, please give the English citation first, followed by the original foreign-language citation. See the end of this document for formats and examples of common references. For a complete discussion of references and their formats, see the IEEE style manual at www.ieee.org/authortools.

\subsection{Footnotes}

Number footnotes separately in superscripts (Insert $\mid$ Footnote).\footnote{It is recommended that footnotes be avoided (except for the unnumbered footnote with the receipt date on the first page). Instead, try to integrate the footnote information into the text.}  Place the actual footnote at the bottom of the column in which it is cited; do not put footnotes in the reference list (endnotes). Use letters for table footnotes (see Table I).
